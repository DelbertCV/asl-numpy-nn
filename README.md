# asl-numpy-nn
Neural network implemented from scratch using numpy as the base to recognize the ASL gestures.

### ASL Dataset

The dataset used in this repo was downloaded from this site:

```
https://www.kaggle.com/datasets/madhavanair/american-sign-language-dataset
```

It is important to have in mind that the letters `J` and `Z` have movement, so, this project is not aimed to recognize these letters.
